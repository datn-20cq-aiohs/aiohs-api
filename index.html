    • vietnam_holiday schema

  • Create Clean_subscription flow:
    • 1. Let user input: from_date (the date will start job), week_dates(working dates in a week), number_of_month
    • 2. Call api: CalculateAllWorkingDates to get all working dates
    • 3. Call api: CalculateCleanSubscriptionPrice with input: working_hour(time to start working in a day), duration_per_date, working_dates(the the response of CalculateAllWorkingDates api)

    • 4. Call calculate price again if user change(rm or add dates)
    • 5. Create order
